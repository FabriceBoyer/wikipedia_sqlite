# Screenshots

This directory contains screenshots of the Wikipedia SQLite web interface.

## Required Screenshots

To add screenshots, please capture the following views:

1. **home.png** - The main search page
2. **search-results.png** - Search results page showing a list of matching articles
3. **article-view.png** - An article view showing the full content
4. **mobile-view.png** - Mobile responsive view (optional)

## How to Take Screenshots

1. Start the server: `go run .`
2. Navigate to http://localhost:9096
3. Take screenshots of:
   - The initial search page
   - A search with results (e.g., search for "Python")
   - An article view (click on any result)
4. Save them in this directory with the names above

## Image Guidelines

- Format: PNG or JPG
- Recommended size: 1920x1080 or similar
- Keep file sizes reasonable (< 500KB each)
- Use descriptive filenames
